# Mamba Framework

## Overview
Mamba Framework is a production-ready Go web framework designed specifically for building enterprise business applications. It prioritizes simplicity, productivity, and rapid CRUD development over flexibility.

## Philosophy
- One page = One handler
- One SQL statement = One operation
- One package = One responsibility
- Keep everything obvious

## Installation
```bash
go mod init your-project
go get github.com/mamba-framework/mamba
